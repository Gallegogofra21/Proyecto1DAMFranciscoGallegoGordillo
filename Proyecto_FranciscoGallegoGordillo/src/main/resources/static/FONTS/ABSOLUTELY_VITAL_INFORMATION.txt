Starella Tattoo is a professional typeface with flowing 
letter forms representing the merge between beauty and attitude. 

The font is provided as a regular version, as well as a version 
decorated with diamonds in each letter. 

Use symbols _ [ ] at any place in a word to create swashes. 
Example: S[tarellaTatt]oo

It supports all Latin-based European languages, contains numbers 
and all symbols you'll ever need.
